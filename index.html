ECMAScript 2015 [ES6] Provides :

  • Let Keyword (allows you to declare a variable with block scope)
  • Const Keyword (allows you to declare a constant value)
  • Arrow Function ()=>{5*6} insteed of function name () {return (5*6)}
  • Loop Over Array
  • Loop Over String
  • String.includes() ===> (true/false)
  • String.startsWith() ===> (true/false)
  • String.endsWith() ===> (true/false)
  • Sets That Returns Unique Values
  • Spread Operators That We Can Convert String To Array [...str]
  • Array.from() That We Can Create Array From String & Sets
